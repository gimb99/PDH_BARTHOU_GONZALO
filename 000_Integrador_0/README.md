# Laboratorio Integrador 0: Procesamiento de Texto Básico

## Descripción

Este trabajo práctico individual introduce conceptos básicos de Procesamiento de Lenguaje Natural (NLP) mediante la implementación de tareas fundamentales: análisis de frecuencia de palabras y generación de resúmenes automáticos. Se utilizan herramientas de Python como `pandas`, `re` y `string` para el procesamiento, y `gradio` para la creación de una interfaz interactiva.

## Objetivo

Desarrollar funciones en Python para procesar texto, calcular la frecuencia de palabras y generar resúmenes automatizados. Se busca comprender los pasos iniciales del procesamiento de texto y construir una aplicación simple para interactuar con los resultados.

## Stack Tecnológico

- **Python**
- **Pandas**
- **Re (Regex)**
- **String**
- **Gradio** (Interfaz)
- **Google Colab** (Entorno de ejecución original)

## Estructura del Proyecto

- `GBG_TP_Integrador_0.ipynb`: Notebook con las actividades de análisis de frecuencia y resumen.
- `README.md`: Este archivo.

## Contenido del Notebook

1.  **Análisis de Frecuencia de Palabras**: Funciones para limpiar texto, contar palabras y mostrar las más frecuentes.
2.  **Generación de Resumen**: Funciones para calcular la importancia de oraciones basadas en frecuencia y formatear un resumen.
3.  **Interfaz Interactiva con Gradio**: Implementación de una interfaz simple para ingresar texto y obtener frecuencias y resumen.

## Instrucciones de Uso

1.  Asegúrate de tener instaladas las dependencias listadas en la sección "Stack Tecnológico".
2.  Abre el notebook `GBG_TP_Integrador_0.ipynb` en Google Colab o en tu entorno local.
3.  Ejecuta las celdas secuencialmente.

## Aprendizajes Clave

- Procesamiento básico de texto (limpieza, tokenización manual).
- Cálculo de frecuencia de palabras.
- Conceptos iniciales de resumen automático basado en frecuencia.
- Creación de interfaces simples con Gradio.
